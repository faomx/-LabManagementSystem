# 🔬 Academic Lab Management System

A Java desktop application developed for my second-year university coursework in **Human-Computer Interaction (IHM)**. This system streamlines the management of members, groups, and laboratory materials across three specialized departments.

## 🚀 Features
- **Departmental Dashboards**: Tailored views for Science & Tech (ST), Computer Science (CS), and Biology (BIO).
- **Member Tracking**: CRUD operations for students and staff with specialty-specific data.
- **Resource Management**: Inventory tracking for laboratory equipment.
- **Persistent Storage**: Full integration with **MySQL** for data reliability.
- **Security**: Secure login system with role-based access control.
